### 2. KEY TAKEAWAYS / ĐIỂM CHÍNH

**[ENGLISH VERSION]**  
- Skills give agents repeatable step-by-step know-how.  
- Progressive disclosure cuts active context by up to 98%.  
- SKILL.md metadata is the most important trigger surface.  
- Trace-driven creation avoids duplicate effort.  
- Compose skills through DAGs and capability profiles.

**[VIETNAMESE VERSION]**  
- Skills cung cấp know-how từng bước lặp lại cho agent.  
- Progressive disclosure cắt active context tới 98%.  
- Metadata SKILL.md là trigger surface quan trọng nhất.  
- Trace-driven creation tránh effort trùng lặp.  
- Compose skills qua DAGs và capability profiles.

### 3. CAPSTONE CONNECTIONS / KẾT NỐI CAPSTONE
