### 2. KEY TAKEAWAYS / ĐIỂM CHÍNH

**[ENGLISH VERSION]**  
- MCP solves the N×M problem with discovery, configuration, connection.  
- A2A enables bounded/unbounded specialist collaboration.  
- A2UI lets agents render generative UI components at runtime.  
- Commerce protocols are optional but relevant for monetization.  
- Use stdio locally, SSE for remote services.

**[VIETNAMESE VERSION]**  
- MCP giải quyết N×M bằng discovery, configuration, connection.  
- A2A cho phép hợp tác chuyên gia bounded/unbounded.  
- A2UI cho agent tạo generative UI tại runtime.  
- Commerce protocols tùy chọn nhưng liên quan đến monetization.  
- Dùng stdio local, SSE cho remote.

### 3. CAPSTONE CONNECTIONS / KẾT NỐI CAPSTONE

**[ENGLISH VERSION]**  
- Map browser actions to MCP-shaped tools.  
- Start with stdio transport for local backend tool execution.  
- Design tool inputs/outputs as JSON schemas.

**[VIETNAMESE VERSION]**  
- Ánh hành động trình duyệt sang tools dạng MCP.  
- Bắt đầu với stdio transport cho local backend tool execution.  
- Thiết kế inputs/outputs tool dạng JSON schema.
